# qodstar-board
qodstar board for monitor

## 技术栈：
  react + redux + webpack + react-router + ES6/7/8 + immutable


## 运行项目（nodejs 6.0+）

```
 git clone git@github.com:catowl/qodstar-board.git

 cd qodstar-board

 npm i  或者运行  yarn(推荐)
  
 npm start

 npm run build （发布）
```
